# VsphereDiagram

A tool to generate diagram for vsphere

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'vsphere_diagram'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vsphere_diagram
